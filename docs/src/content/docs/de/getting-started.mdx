## Voraussetzungen

Du musst eine Starlight-Website eingerichtet haben.
Wenn du noch keine hast, kannst du die Anleitung [„Erste Schritte“](https://starlight.astro.build/de/getting-started/) in den Starlight-Dokumenten befolgen, um eine zu erstellen.

## Installation

import { Steps, Tabs, TabItem } from "@astrojs/starlight/components";

<Steps>

1. `starlight-cooler-credit` ist ein Starlight [Plugin](https://starlight.astro.build/de/reference/plugins/). Installiere es, indem du den folgenden Befehl in deinem Terminal ausführst:

   <Tabs syncKey="pkg">

   <TabItem label="npm">

   ```sh
   npm install starlight-cooler-credit
   ```

   </TabItem>

   <TabItem label="pnpm">

   ```sh
   pnpm add starlight-cooler-credit
   ```

   </TabItem>

   <TabItem label="Yarn">

   ```sh
   yarn add starlight-cooler-credit
   ```

   </TabItem>

   </Tabs>

2. Konfiguriere das Plugin in deiner Starlight [Konfiguration](https://starlight.astro.build/de/reference/configuration/#plugins) in der Datei `astro.config.mjs`.

   ```diff lang="js"
   // astro.config.mjs
   import starlight from '@astrojs/starlight'
   import { defineConfig } from 'astro/config'
   +import starlightCoolerCredit from 'starlight-cooler-credit'

   export default defineConfig({
     integrations: [
       starlight({
   +      plugins: [starlightCoolerCredit()],
         title: 'My Docs',
       }),
     ],
   })
   ```

3. [Starte den Entwicklungsserver](https://starlight.astro.build/getting-started/#start-the-development-server), um das Plugin in Aktion zu sehen.

</Steps>

Das Verhalten des Starlight Coolere Anerkennung Plugins kann mit verschiedenen [Konfigurationsoptionen](/de/configuration) angepasst werden.
